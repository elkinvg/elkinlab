Запуск задач Livni

NOTES:
1) Нужно авторизоваться на livni.jinr.ru
2) Удалять можно только свои jobs, но мега-admin (Г.А.) может удалять все.
3) Для создания новой job ("from scratch") - достаточно выбрать "Task type filter". Или кликнув "Create new", когда выведена инфа и таблица параметров по выбранной job. Для создания job по шаблону - достаточно кликнуть по одной из списка справа, отредактировать параметр(ы) и нажать "Start".
4) Фильтры для вывода списка - самоочевидны.
5) Вывод результатов - по кнопкам над таблицей параметров. Для возврата к списку - давим кнопку "Refresh".
6) Для отключения тултипов - в правом нижнем углу страницы есть сылка "help off".
