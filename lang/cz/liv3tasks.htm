Создание задач Livni

NOTES:
1) Список людей, которым можно создавать и курочить задачи - пока захардкодил:
Шелков, Гуськов, Жемчугов, Демичев, т.к. неясно, по какому признаку разрешать/запрещать.
Все остальные могут потыркать в режиме View Only
2) Задачи с Popularity > 0 не помечаются чекбоксом для удаления. Т.е. убить задачу с параметрами можно только предварительно обнулив Popularity. Т.е. неслучайно.
3) Для создания новой задачи (почти from scratch) - Refresh, Create New
4) Для создания новой задачи (по готовому шаблону) - Refresh, быбрать из списка, Create New
5) Для модификации любой задачи - Refresh, быбрать из списка, Save changes
6) Синхронизация application и параметров - на совести создателей задач.
7) Для отключения тултипов - в правом нижнем углу страницы есть сылка "help off".
