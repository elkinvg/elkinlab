Журнал Livni

NOTES:
1) Эксперты и админ может редактировать журнал, остальные - только просматривать
2) Для добавления новой записи в журнал (from scratch) - Refresh, Create New
3) Для добавления новой записи по шаблону - Refresh, быбрать из списка, Create New
4) Для модификации любой записи - Refresh, быбрать из списка, Save changes
5) Для отключения тултипов - в правом нижнем углу страницы есть сылка "help off".
