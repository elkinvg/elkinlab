Запуск заданий Livni

  • Для создания новой job ("from scratch") - достаточно выбрать "Task type filter".
    Или кликнуть "Create new", когда по выбрана job из списка справа.
  • Для создания job по шаблону - достаточно кликнуть по одной из списка справа, отредактировать параметр(ы) и нажать "Start".
  • Используйте фильтры для сужения списка заданий.
  • Вывод графиков - по кнопкам над таблицей параметров. Для возврата к списку - нажмите "Refresh".
  • Удалять можно только свои jobs, но администратор может удалять любые.
  • Для отключения тултипов - кликните по ссылке "help off" в правом нижнем углу страницы.
