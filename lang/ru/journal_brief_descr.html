Журнал установки предназначен для фиксации условий работы станций: изменения порогов, высокого напряжения, отключения и подключения оборудования и т.д. Журнал ведется только экспертами, но просмотр разрешен, и, более того, рекомендован всем участникам проекта, анализирующим физические данные установки.

Ведение журнала установки "Русалка"

  • Для добавления новой записи в журнал ("from scratch") - нажмите "Refresh", затем - "Create New".
  • Для добавления новой записи по шаблону - нажмите "Refresh", выберите запись из списка, затем - "Create New".
  • Для модификации любой записи (только эксперты и администратор) - нажмите "Refresh", выберите запись из списка, затем - "Save changes".
  • Для отключения тултипов - кликните по ссылке "help off" в правом нижнем углу страницы.
