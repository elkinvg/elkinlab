Задачи (типы заданий) - это предопределенные, подготовленные и апробированные алгоритмы для решения относительно узкого спектра задач. Вариабельность результатов определяется набором параметров, которые вы в определенных пределах можете менять. Задачи подготавливаются некоторыми из экспертов, поэтому остальные - могут только посмотреть список. Например, для выяснения уровня сложности: для новичков (lev=0), для продвинутых(lev=1) и для экспертов(lev=2). Продвигайтесь от простого к сложному.

Создание задач Livni

  • Для создания новой задачи (почти "from scratch") - нажмите "Refresh", затем-"Create New"
  • Для создания новой задачи по шаблону - нажмите "Refresh", выберите из списка, затем - "Create New"
  • Для модификации любой задачи - нажмитье "Refresh", выберите из списка, внесите изменения, затем - "Save changes"
  • Создатели задач обеспечивают однозначное соответствие (синхронизацию) параметров и их интерпретацию со стороны application.
  • Задачи с "Popularity" > 0 не помечаются чекбоксом для удаления.
    Т.е. удалить задачу с параметрами можно только намеренно, предварительно обнулив "Popularity".
  • Для отключения тултипов - кликните по ссылке "help off" в правом нижнем углу страницы.
